### Scenario: Access log fires when a handler raises
**Area**: `app/api/middleware/request_context.py`
**Type**: functional / observability regression
**Preconditions**: backend running locally with `LOG_LEVEL=DEBUG`.
**Steps**:
1. Add a throwaway route that raises (`@app.get("/boom")\ndef _(): raise RuntimeError("k")`).
2. `curl -i http://127.0.0.1:8000/boom`.
3. Tail the server logs.
**Expected Result**: response is `500 Internal Server Error`; logs contain
exactly **one** `http_request` line with `status_code=500`, a non-empty

line from the global handler appears in addition to (not instead of)
the access log.
**Verification**: `rg http_request app.log | rg /boom` returns a single match.
